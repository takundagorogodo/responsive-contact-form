# Responsive Contact Form

This project is a responsive contact form designed using **HTML5 and CSS3**.  
It features a clean layout with contact information on the left and a message form on the right.

## Features
- Responsive design for desktop and mobile
- Flexbox-based layout
- Clean and modern UI
- Google Fonts integration
- Font Awesome icons
- No frameworks or libraries required

## Technologies Used
- HTML5
- CSS3 (Flexbox & Media Queries)

## Purpose
This project was created to practice responsive web design and UI structuring using pure HTML and CSS.

## Future Improvements
- Form validation using JavaScript
- Backend integration (Node.js / PHP)
- Improved accessibility with labels and ARIA attributes

## Preview
You can clone the repository and open `contactform.html` in your browser to view the project.
